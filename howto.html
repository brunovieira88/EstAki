Como Utilizar

1. Escolhe o cacifo

Escolhe o cacifo onde queres receber a encomenda, entre os locais disponíveis.

2. Encomenda depositada

A encomenda é colocada no cacifo selecionado para ti.

3. Recebe o PIN

Recebe o PIN para abrir o cacifo, diretamente no teu telemóvel ou e-mail.

4. Insere o PIN

Insere o PIN no cacifo e abre-o para recolher a tua encomenda.
